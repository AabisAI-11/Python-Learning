# FILE HANDLING PROJECT (PYTHON):-
A simple menu-driven File Handling system built in Python 
This project performs basic CRUD operations on files and helped me understand how file management works internally.

>>Features

.Create a file
.Read file content
.Update file (Rename / Overwrite / Append)
.Delete a file
.Displays available files before every operation

>>Concepts Used

. pathlib module
. File modes (r, w, a)
. Exception handling (try-except)
. Menu-driven program logic
. Basic CRUD operations

>>Purpose:-

Built as part of my Python learning journey during B.Tech to strengthen my understanding of:

.File operations

.Path handling

.Error handling

.Program structure
